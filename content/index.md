# 欢迎来到我的数字花园

这是一个基于Quartz和Obsidian的数字花园，用于记录和分享我的知识、想法和经验。

## 关于这个花园

- **技术栈**：Quartz + Obsidian + GitHub Pages
- **内容**：技术文章、项目经验、个人思考、资源链接
- **更新频率**：持续更新

## 如何使用

1. **浏览内容**：使用左侧菜单导航到不同的内容分类
2. **搜索**：使用顶部搜索栏查找特定内容
3. **Obsidian同步**：通过Obsidian编辑内容，然后推送到GitHub更新网站
4. **查看指南**：阅读[使用指南](/docs/usage-guide/)了解详细操作方法
